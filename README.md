### Ortholinear, columnar stagger keyboard switch plate and case
* 40%, non-split layout designed to easy to use and ergonomic
* Compatible with Ergodox style keysets


### Assembly:
* Intended to be used with a metal plate sandwich mounted inside the case, fastened with screws.
* The rear cutout is designed for a mini usb connection, like the one on the Teensy2.0.


### Note:
* STL files may be misoriented for printing, thoroughly inspect files before continuing to print.
* Default firmware does not include any LED configuration.
